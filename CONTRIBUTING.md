# Contributing to YouTube-Downloader

Thank you for considering contributing!

## How to contribute

1. Fork the repository
2. Create a new branch:
   git checkout -b feature-name
3. Make your changes
4. Commit using conventional commits:
   feat: added playlist UI
   fix: resolved path issue
5. Push and open a Pull Request

## Coding Style
- Follow Python PEP 8 conventions
- Use descriptive variable names
- Write clean, readable functions
- Avoid unnecessary global vars

## Issues
- Use the GitHub Issue Tracker
- Provide steps to reproduce
- Include log output if possible

## Pull Requests
- One feature per PR
- Keep PRs small and readable
- Tests recommended when possible

Thanks for helping make this project better!
